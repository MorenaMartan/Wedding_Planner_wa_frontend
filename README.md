# Wedding Planner Frontend

Frontend for a wedding planning platform designed to centralize guest management, budgeting, timelines, and vendor coordination.

## Status

In active development.

## Features

- Guest management interface
- Budget and planning views
- Vendor coordination structure
- Responsive Vue frontend
- Prepared for backend API integration

## Tech Stack

Vue.js, JavaScript, Vite, HTML, CSS
